HOW TO ADD A NEW EVENT PHOTO
1. Save photo as: YYYY-MM-description-NN.jpg
   Example: 2026-09-business-visit-01.jpg
2. Copy the event card template from Dunsin_Website_Handoff.md
3. Fill in: category, filename, date, title, location, description
4. Paste new card at TOP of .events-grid in community.html
5. Update the 3 homepage community snapshot images
